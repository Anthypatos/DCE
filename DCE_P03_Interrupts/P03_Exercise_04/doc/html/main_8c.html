Detailed Description

The state of a pushbutton is shown in a LED.

Author
Juan de la Cruz Caravaca Guerrero
Date
13/04/2022
Description:
The LED and the PUSHBUTTON are connected to the PORT specified in "PORTs.h". A RED_LED is switch ON/OFF by means a push button while a GREEN_LED is always blinking. In this version, the pushbutton is connected to an external interrupt.
